---

Infisical supports connecting to Hasura Cloud using a [Personal Access Token](https://hasura.io/docs/2.0/hasura-cloud/account-settings/#access-tokens) to manage your project's environment variables.

## Create a Hasura Cloud Personal Access Token

<Steps>
  <Step title="Open your Hasura Cloud account settings">
    Log in to [Hasura Cloud](https://cloud.hasura.io), click your profile icon in the top-right corner, and select **Account Settings**.

    ![Hasura Cloud Account Settings](/images/app-connections/hasura-cloud/hasura-cloud-account-settings.png)
  </Step>
  <Step title="Navigate to Access Tokens">
    In the account settings, open the **Access Tokens** section.

    ![Hasura Cloud Access Tokens](/images/app-connections/hasura-cloud/hasura-cloud-access-tokens.png)
  </Step>
  <Step title="Create a new Personal Access Token">
    Provide a descriptive name for the token, then click **Create**.

    ![Create Hasura Cloud Token](/images/app-connections/hasura-cloud/hasura-cloud-create-token.png)
  </Step>
  <Step title="Save the token">
    After creating the token, copy its value and store it securely for the next steps. You will not be able to view it again.

    ![Hasura Cloud Token Created](/images/app-connections/hasura-cloud/hasura-cloud-token-created.png)
  </Step>
</Steps>

## Create a Hasura Cloud Connection in Infisical

<Tabs>
  <Tab title="Infisical UI">
    <Steps>
      <Step title="Navigate to App Connections">
        In your Infisical dashboard, navigate to the **Integrations** tab in the desired project, then select **App Connections**.

        ![App Connections Tab](/images/app-connections/general/add-connection.png)
      </Step>
      <Step title="Select Hasura Cloud Connection">
        Click **+ Add Connection** and choose **Hasura Cloud Connection** from the list of integrations.

        ![Select Hasura Cloud Connection](/images/app-connections/hasura-cloud/app-connection-option.png)
      </Step>
      <Step title="Fill out the Hasura Cloud Connection form">
        Complete the form by providing:
        - A descriptive name for the connection
        - An optional description
        - The Personal Access Token from the previous step

        ![Hasura Cloud Connection Modal](/images/app-connections/hasura-cloud/app-connection-form.png)
      </Step>
      <Step title="Connection created">
        After submitting the form, your **Hasura Cloud Connection** will be successfully created and ready to use with your Infisical project.

        ![Hasura Cloud Connection Created](/images/app-connections/hasura-cloud/app-connection-generated.png)
      </Step>
    </Steps>
  </Tab>

  <Tab title="API">
    To create a Hasura Cloud Connection via API, send a request to the [Create Hasura Cloud Connection](/api-reference/endpoints/app-connections/hasura-cloud/create) endpoint.

    ### Sample request

    ```bash Request
    curl    --request POST \
            --url https://app.infisical.com/api/v1/app-connections/hasura-cloud \
            --header 'Content-Type: application/json' \
            --data '{
                "name": "my-hasura-cloud-connection",
                "method": "access-token",
                "projectId": "7ffbb072-2575-495a-b5b0-127f88caef78",
                "credentials": {
                    "accessToken": "[PERSONAL ACCESS TOKEN]"
                }
            }'
    ```
