Infisical supports connecting to GitLab using [Access Tokens](https://docs.gitlab.com/user/profile/service_accounts/#create-a-personal-access-token-for-a-service-account).

## Get a GitLab Access Token



<Steps>
    <Step title="Navigate to Access Tokens">
        In GitLab, navigate to the **Access Tokens** page for the Service Account, User, Project or Group you want to create an access token for under **Settings** > **Access Tokens**.

        <Note>
            While Infisical supports Personal, Project and Group access tokens, we recommend creating a designated [Service Account](https://docs.gitlab.com/user/profile/service_accounts/#create-a-personal-access-token-for-a-service-account) for Infisical.
        </Note>

        ![GitLab Access Tokens](/images/app-connections/gitlab/gitlab-add-access-token.png)
    </Step>
    <Step title="Create Access Token">
        Give your token a name and optionally set an expiry date.

        <Note>
            If you configure an expiry date for your access token, you will need to manually rotate to a new token before the expiration date to prevent service interruption.
        </Note>

        Depending on your use-case, select the appropriate scopes for your token:
        <Tabs>
            <Tab title="Secret Scanning">
               Add the `api` scope to allow Infisical to list projects, create webhooks, and scan repositories for secret leaks.

               ![GitLab Create Token](/images/app-connections/gitlab/gitlab-secret-scanning-token.png)
            </Tab>
        </Tabs>

        Scroll down and click **Create access token**.
    </Step>
    <Step title="Copy Access Token">
        Copy your access token and save it for the steps below.

        ![GitLab Copy Token](/images/app-connections/gitlab/gitlab-copy-token.png)
    </Step>
</Steps>


## Setup GitLab Connection in Infisical

<Tabs>
    <Tab title="Infisical UI">
        <Steps>
            <Step title="Navigate to App Connections">
                In your Infisical dashboard, go to **Organization Settings** and select the **App Connections** tab.
                ![App Connections Tab](/images/app-connections/general/add-connection.png)
            </Step>
            <Step title="Add Connection">
                Click the **+ Add Connection** button and select the **GitLab Connection** option.
                ![Select GitLab Connection](/images/app-connections/gitlab/select-gitlab-connection.png)
            </Step>
            <Step title="Configure Connection">
                Configure your GitLab Connection using the access token generated in the steps above. Then click **Connect to Windmill**.
                ![GitLab Configure Connection](/images/app-connections/gitlab/create-gitlab-access-token-connection.png)

                - **Name**: The name of the connection to be created. Must be slug-friendly.
                - **Description**: An optional description to provide details about this connection.
                - **Instance URL**: The URL of your GitLab instance. If you are not self-hosting GitLab you can leave this field blank.
                - **Access Token**: The access token generated in the steps above.
            </Step>
            <Step title="Connection Created">
                Your GitLab Connection is now available for use.
                ![Windmill GitLab Created](/images/app-connections/gitlab/gitlab-access-token-connection-created.png)
            </Step>
        </Steps>
    </Tab>
    <Tab title="API">
        To create a GitLab Connection, make an API request to the [Create GitLab
        Connection](/api-reference/endpoints/app-connections/gitlab/create) API endpoint.

        ### Sample request

        ```bash Request
        curl    --request POST \
                --url https://app.infisical.com/api/v1/app-connections/gitlab \
                --header 'Content-Type: application/json' \
                --data '{
                    "name": "my-gitlab-connection",
                    "method": "access-token",
                    "credentials": {
                        "token": "...",
                        "instanceUrl": "https://www.gitlab.com"
                    }
                }'
        ```

        ### Sample response

        ```bash Response
        {
            "appConnection": {
                "id": "3c90c3cc-0d44-4b50-8888-8dd25736052a",
                "name": "my-gitlab-connection",
                "version": 123,
                "orgId": "3c90c3cc-0d44-4b50-8888-8dd25736052a",
                "createdAt": "2025-04-01T05:31:56Z",
                "updatedAt": "2025-04-01T05:31:56Z",
                "app": "gitlab",
                "method": "access-token",
                "credentials": {
                    "instanceUrl": "https://www.gitlab.com"
                }
            }
        }
        ```
    </Tab>
</Tabs>
