---
title: "Trigger.dev Connection"
description: "Learn how to configure a Trigger.dev Connection for Infisical."
---

Infisical supports the use of Trigger.dev API tokens (secret keys or personal access tokens) to connect with Trigger.dev.

<Note>
  Trigger.dev secret keys are environment-specific. Make sure the key you use matches the environment you intend to sync (dev, staging, or prod).
</Note>

## Create a Trigger.dev API Token

<Steps>
  <Step title="Open your Trigger.dev project settings">
    In the Trigger.dev dashboard, open your project settings and navigate to **API Keys**.
  </Step>
  <Step title="Create a new API token">
    Create a new key and copy it securely. You will need it to create the connection in Infisical.
  </Step>
</Steps>

## Create a Trigger.dev Connection in Infisical

<Tabs>
  <Tab title="Infisical UI">
    <Steps>
      <Step title="Navigate to App Connections">
        In your Infisical dashboard, navigate to the **App Connections** page in the desired project.

        ![App Connections Tab](/images/app-connections/general/add-connection.png)
      </Step>
      <Step title="Select Trigger.dev Connection">
        Click **+ Add Connection** and choose **Trigger.dev Connection** from the list of integrations.
      </Step>
      <Step title="Fill out the Trigger.dev Connection form">
        Complete the form by providing:
        - A descriptive name for the connection
        - An optional description
        - The **API Token** from Trigger.dev
        - (Optional) **API URL** if you are using a self-hosted Trigger.dev instance
      </Step>
      <Step title="Connection created">
        After submitting the form, your **Trigger.dev Connection** will be successfully created and ready to use with your Infisical project.
      </Step>
    </Steps>
  </Tab>

  <Tab title="API">
    To create a Trigger.dev Connection via API, send a request to the [Create Trigger.dev Connection](/api-reference/endpoints/app-connections/trigger-dev/create) endpoint.

    ### Sample request

    ```bash Request
    curl    --request POST \
            --url https://app.infisical.com/api/v1/app-connections/trigger-dev \
            --header 'Content-Type: application/json' \
            --data '{
                "name": "my-trigger-dev-connection",
                "method": "api-token",
                "projectId": "7ffbb072-2575-495a-b5b0-127f88caef78",
                "credentials": {
                    "apiToken": "[API TOKEN]",
                    "apiUrl": "https://api.trigger.dev"
                }
            }'
    ```
