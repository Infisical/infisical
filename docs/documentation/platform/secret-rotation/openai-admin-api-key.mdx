---
title: "OpenAI Admin API Key"
description: "Learn how to automatically rotate OpenAI admin API keys."
---

<Note>
  **Rotation Type: Dual-Phase**

  This rotation maintains two active credential sets with overlapping validity, ensuring zero-downtime during rotation cycles.
</Note>

## Prerequisites

- Create an [OpenAI Connection](/integrations/app-connections/openai) using an **Admin API key**. That connection is used to create and delete admin API keys on your behalf during rotation.

## Create an OpenAI Admin API Key Rotation in Infisical

<Tabs>
    <Tab title="Infisical UI">
        1. Navigate to your Secret Manager Project's Dashboard and select **Add Secret Rotation** from the actions dropdown.

        ![Secret Manager Dashboard](/images/secret-rotations-v2/generic/add-secret-rotation.png)

        2. Select the **OpenAI Admin API Key** option.

        ![Select OpenAI Admin API Key](/images/secret-rotations-v2/openai-admin-api-key/select-openai-admin-api-key.png)

        3. Configure the rotation behavior, then click **Next**.

        ![Rotation Configuration](/images/secret-rotations-v2/openai-admin-api-key/configuration.png)

        - **OpenAI Connection** – The connection (with an Admin API key) that will create and delete admin API keys during rotation.
        - **Rotation Interval** – The interval, in days, after which a rotation is triggered.
        - **Rotate At** – The local time of day when rotation runs once the interval has elapsed.
        - **Auto-Rotation Enabled** – Whether to rotate automatically on the interval. Turn off to rotate only manually or pause rotation.

        4. Set the OpenAI Admin API Key parameters, then click **Next**.

        ![Rotation Parameters](/images/secret-rotations-v2/openai-admin-api-key/parameters.png)

        - **Key name** – The name for the generated OpenAI admin API key (required, up to 100 characters). Infisical appends a timestamp suffix to this name in the OpenAI dashboard (e.g. `my-app-key-1720268400000`) so that the active and previous keys stay distinguishable during the overlap window. Keys are deleted by their key ID, so the name is used only for identification.

        5. Specify the secret name that the rotated API key will be mapped to. Then click **Next**.

        ![Rotation Secrets Mapping](/images/secret-rotations-v2/openai-admin-api-key/secrets-mapping.png)

        - **API Key** – The name of the secret in Infisical where the rotated admin API key value will be stored.

        6. Give your rotation a name and description (optional). Then click **Next**.

        ![Rotation Details](/images/secret-rotations-v2/openai-admin-api-key/details.png)

        - **Name** – A slug-friendly name for this rotation configuration.
        - **Description** (optional) – Notes about this rotation.

        7. Review your configuration, then click **Create Secret Rotation**.

        ![Rotation Review](/images/secret-rotations-v2/openai-admin-api-key/review.png)

        8. Your **OpenAI Admin API Key** rotation is created. The current admin API key is available as a secret at the mapped path. Rotations will create a new key, switch the active secret to it, then revoke the previous key for zero-downtime rotation.

        ![Rotation Created](/images/secret-rotations-v2/openai-admin-api-key/created.png)
    </Tab>
    <Tab title="API">
        To create an OpenAI Admin API Key rotation, call the [Create OpenAI Admin API Key Rotation](/api-reference/endpoints/secret-rotations/openai-admin-api-key/create) API endpoint.

        ### Sample request

        ```bash Request
        curl --request POST \
          --url https://us.infisical.com/api/v2/secret-rotations/openai-admin-api-key \
          --header 'Content-Type: application/json' \
          --data '{
            "name": "my-openai-rotation",
            "projectId": "<project-id>",
            "description": "OpenAI admin API key rotation",
            "connectionId": "<openai-connection-id>",
            "environment": "dev",
            "secretPath": "/",
            "isAutoRotationEnabled": true,
            "rotationInterval": 30,
            "rotateAtUtc": {
              "hours": 0,
              "minutes": 0
            },
            "parameters": {
              "name": "my-app-key"
            },
            "secretsMapping": {
              "apiKey": "OPENAI_ADMIN_API_KEY"
            }
          }'
        ```

        ### Sample response

        ```bash Response
        {
          "secretRotation": {
            "id": "<rotation-id>",
            "name": "my-openai-rotation",
            "description": "OpenAI admin API key rotation",
            "secretsMapping": {
              "apiKey": "OPENAI_ADMIN_API_KEY"
            },
            "isAutoRotationEnabled": true,
            "activeIndex": 0,
            "connectionId": "<openai-connection-id>",
            "rotationInterval": 30,
            "rotateAtUtc": { "hours": 0, "minutes": 0 },
            "type": "openai-admin-api-key",
            "parameters": {
              "name": "my-app-key"
            }
          }
        }
        ```
