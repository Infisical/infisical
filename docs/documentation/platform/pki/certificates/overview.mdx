To issue certificates with Infisical, you need:

1. **Certificate Authority** — A CA to issue certificates from. See [Certificate Authorities](/documentation/platform/pki/ca/overview).
2. **Certificate Policy** — Rules for what certificates can contain. See [Certificate Policies](/documentation/platform/pki/settings/policies).
3. **Certificate Profile** — Combines a CA with a policy. See [Certificate Profiles](/documentation/platform/pki/settings/profiles).
4. **Application** — The resource where teams issue certificates. See [Applications](/documentation/platform/pki/applications/overview).
