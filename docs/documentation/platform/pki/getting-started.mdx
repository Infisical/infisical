## Who manages what?

| Role | Responsibilities |
|------|------------------|
| **Platform Admin** | Set up CAs, policies, profiles. Create Applications and assign team members. |
| **Application Admin** | Configure enrollment methods, alerting, syncs within their Application. |
| **Application Operator** | Issue, renew, and revoke certificates. |

<Info>
  **Platform Admin?** Start with [Certificate Authorities](/documentation/platform/pki/ca/overview), then create [Applications](/documentation/platform/pki/applications/overview) for your teams.
  
  **Team member?** Ask your platform admin to create an Application and assign you.
</Info>
