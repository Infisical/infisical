The Infisical standalone version combines all the essential components of the application into a single container, making deployment and management more straightforward than using Kubernetes or Docker Compose. You will need to have deployments of MongoDB and Redis that you can connect the Infisical instance to. See the podman example for a setup that runs all 3 containers isolated within a pod.

Since all the components are bundled into one image, running this version of Infisical requires a minimum of **230MB of memory**.

This guide assumes you have basic knowledge of Docker and have it installed on your system. If you don't have Docker installed, please follow the official installation guide: https://docs.docker.com/get-docker/

## Pull the Infisical Docker image

Open your terminal or command prompt and enter the following command to pull the Infisical Docker image:

```
docker pull infisical/infisical:latest
```

## Run with podman

Podman is a docker alternative, and is the Red Hat supported container engine on RHEL.

This example shows how to run simple test deployments of MongoDB, Redis and Infisical containers within a pod where only Infisical is exposed to the outside.

First, we create the container and set up the mapping of the external port 80 to port 80 inside the pod. You can replace the external port by chenging 80:80 to something like 1080:80. You will then have to specify the correct port when connecting to your Infisical instance afterwards.

```bash
podman pod create --name infisical_vault -p 80:80
```

Then we create the Redis and MongoDB containers inside this pod. Note that there are no port specifications, as containers in a pod are visible to each other.

```bash
podman run -d --restart=always --pod=infisical_vault --name=infisical-redis docker.io/redis/redis:latest
podman run -d --restart=always --pod=infisical_vault --name=infisical-mongo docker.io/mongodb/mongodb-community-server:latest
```

</Warning>

## Verify the installation:
Once the container is running, open a web browser and navigate to http://localhost:80. That's it! You have successfully installed the Infisical application using a single Docker image.

<Info>
Once installation is complete, you will have to create the first account. No default account is provided.
</Info>
